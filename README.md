## 📖 Description

StudyBuddy is an AI-powered learning platform designed to help students improve their learning experience through:

* Smart study sessions
* Quiz generation
* Progress tracking
* Streak system

## ✨ Features

* 📊 Dashboard with analytics
* 🧠 Quiz system
* 🔥 Streak tracking
* 🎯 Personalized learning topics
